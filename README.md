# DEVLOG blog system
> CRUD with fastapi practice #1

Blog system for developer to log there achievement or todo list

## 🛠️ Stacks
- Vite React(Frontend)
- Fastapi(Backend)
- Supabase(Database)

## ✨ Features
- Upload writing form with CRUD
- Edit posts
- Delete posts

## 🚀 Execute
> Frontend

```
npm install
npm run dev
```

> Backend

Set `.env`
```
pip install -r requirements.txt
uvicorn main:app --reload
```
